Installing the FlowMiner Eclipse plugin is easy!

Installation from Update Site

Follow the steps below to install FlowMiner from our Eclipse update site.

  1. Start Eclipse, then select Help > Install New Software.
  2. Click the Add... button in the top-right.
  3. In the Add Repository dialog that appears, enter FlowMiner for the Name, enter https://powerofpi.github.io/FlowMiner/updates/ for the Location, and click OK
  4. In the Work With dropdown, select FlowMiner.the checkbox next to "FlowMiner" and click Next followed by OK.
  5. Check the box next to "FlowMiner", then click Next and OK. Dependencies will be auto-selected for you.
  6. In the next window, you'll see a list of the plugins to be installed. Click Next.
  7. Read and accept the license agreements, then click Finish. If you get a security warning saying that the authenticity or validity of the software can't be established, click OK.
  8. When the installation completes, restart Eclipse.
FlowMiner requires (and auto-installs) the Atlas analysis platform. If this is the first time you have installed Atlas, you will need to request a license from EnSoft Corp. A free academic license is available, as well as a free trial license. When you fill out the form, please mention FlowMiner.
